#NeuroImage2018
Dynamic low frequency EEG phase synchronization patterns during proactive control of task switching

The raw data can be download from the following link: 
https://drive.google.com/open?id=1hHn1e1sabQTEOor-wt5g2aGK4nG0f9u1

The ITPC computation can be done from the itpc_taskswitching.m file. 

The connectivity matrices (5D) obtained are: 
switch.mat ---> (freq bands(5)x ROIs (68) x ROIs (68) x windows(10)x subjects)
repeat.mat ---> (freq bands(5)x ROIs (68) x ROIs (68) x windows(10)x subjects)
